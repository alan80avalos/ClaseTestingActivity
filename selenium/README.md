# Pruebas con Behave y Selenium

Este proyecto contiene pruebas automatizadas usando **BDD**, **DDT**, y **Selenium**.

La prueba abre Google, busca una universidad, entra al resultado correspondiente y después realiza una búsqueda relacionada dentro del dominio de la universidad.
---
## Requisitos
Antes de ejecutar las pruebas, asegúrate de tener instalado:
- Python 3
- Google Chrome
- pip
---

## Instalación

1. Clonar o descomprimir el proyecto.

2. Entrar a la carpeta del proyecto:

3. Una vez dentro crear una terminal y se recomienda crear un entorno virtual
```
py -m venv venv
venv\Scripts\activate
```

4. Instalar selenium y behave para correr
```
pip install behave selenium
```

5. Ejecutrar con
```
behave
```
